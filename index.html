Git Quiz:

What is Git and why is it so important?

Git is version control system used for tracking changes in files and projects. Git is important as it allows us to create a record of whats been done and revert those changes if needed. It also allows for multiple people to colaborate on files and projcets.

What’s a branch and why do we create them?

A branch is a copy of the master code, we use them so we are able to make changes and work on new features without effecting the master branch.

What’s the working tree?

The working tree is the directory that contains the files you are currently working

What’s the staging area?

The staging area is where files are saved/tracked by git ready to be commited

What’s the command to retrieve the latest version of your files from the remote serve?

git pull origin master/main
